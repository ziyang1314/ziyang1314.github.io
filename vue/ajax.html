ajax和axios、fetch的区别

参考部分

自己主要记住的内容；

  • ajax是对原生xhr（xmlhttprequest）封装，jquery ajax是jquery中提供的实现，所以仅使用发请求这一项的话，需要引用整个jquery；
  • axios 是基于promise对于浏览器和nodejs的http；其实际也是对xhr的封装的promise实现；简洁、体积小，支持并发
  • fetch基于ajax 使用promise实现，优点简洁，支持async和await，但是400和500都会被请求成功，有bug；

axios实际使用

参考部分

个人需要理解和记住的内容

  • 安装或引入axios：npm install axio
  • 常规使用axios： 
                post:
            axios({
                method: 'post',
                url: '/user/12345',
                data: {
                  firstName: 'Fred',
                  lastName: 'Flintstone'
                }
              });
              post:
              axios.post('/user', {
                firstName: 'Fred',
                lastName: 'Flintstone'
              })
              .then(function (response) {
                console.log(response);
              })
              .catch(function (error) {
                console.log(error);
              });
            默认是get；
            也可以
            axios.get(url,{params:{}})
            url后也可以直接跟参数，和ajax区别是参数不是data而是params；
